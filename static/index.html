Shenmeci is an open-source project developed by Rodolfo Carvalho. The code is hosted at GitHub.

The development began as a final project for the Language and Information Processing class at Beijing Institute of Technology (北京理工大学).

Under the hood, the project is about constructing a simple, fast and enough accurate Chinese text sementation engine. Simple means it is easy to use and understand the implementation. Fast means that the user should have a feeling of instantaneous results. Enough accurate means that the correctness of the segmentation is enough to provide useful Chinese-English translations.

For providing translations it uses the Creative Commons licensed Chinese-English dictionary CC-CEDICT.
